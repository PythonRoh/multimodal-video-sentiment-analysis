# Multimodal Video Sentiment Analysis – Frontend

A modern AI-powered frontend interface for the Multimodal Video Sentiment Analysis platform.

This frontend allows users to upload videos, analyze emotions and sentiments from multimedia content, and visualize AI-generated insights through an interactive and responsive UI.

---

## 🚀 Features

- Modern responsive UI
- Video upload interface
- AI sentiment analysis visualization
- Emotion detection dashboards
- Interactive analytics and charts
- SaaS-inspired clean design
- Fast and scalable frontend architecture

---

## 🛠️ Tech Stack

- React.js / Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite / Next Build System
- REST API Integration
